Well known examples for classification (with unsupervised learning)

Lets do it by performing k-means clustering on the MNIST pixel features an see if we can identify unique clusters of digits without using the response variable. Here, we declare \(k = 10\) only because we already know there are 10 unique digits represented in the data.
